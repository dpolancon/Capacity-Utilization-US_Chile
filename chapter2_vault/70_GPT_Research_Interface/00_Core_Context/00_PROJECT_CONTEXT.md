# Project Context

Chapter 2 reconstructs productive capacity, capacity utilization, accumulation, distribution, and profitability for the United States and Chile.

The project treats productive capacity as latent and historically variable. It combines national accounts, fixed-asset measurement, GPIM reconstruction, integration diagnostics, long-run estimation, and historical interpretation.

The current repo is an active research system, not a static paper folder. It contains data pipelines, validation gates, methods notes, writing materials, and advisor-facing artifacts.
