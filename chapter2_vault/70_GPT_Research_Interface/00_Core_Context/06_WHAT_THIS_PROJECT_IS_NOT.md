# What This Project Is Not

This is not:

- a conventional production-function exercise;
- a trend-filter estimate of potential output;
- a direct replication of Shaikh;
- a VECM-first project;
- a generic capital-output-ratio study;
- a model in which autonomous demand directly creates productive capacity;
- a simple U.S.–Chile panel;
- an exercise where statistical consistency substitutes for theoretical identification.
