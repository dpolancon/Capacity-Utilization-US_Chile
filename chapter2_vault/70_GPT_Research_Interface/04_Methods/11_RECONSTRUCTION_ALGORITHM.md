# Reconstruction Algorithm

1. Read authoritative coefficient estimates.
2. Compute the state-dependent \(\widehat{\theta}_t\).
3. Multiply by capital growth or apply the accepted composition identity.
4. Integrate to obtain the capacity-growth path.
5. Set the level using the selected anchor.
6. Compute utilization.
7. Validate signs, continuity, anchor exactness, and sensitivity.
8. Compare Specification A and B.
