# CPR Method Note

CPR is the local label for the interactive or polynomial cointegration architecture used to handle integrated state variables and interactions.

The method note must identify:

- orders of integration;
- polynomial cointegrating relations;
- estimable coefficient combinations;
- inference procedure;
- relation to earlier FM-OLS estimates;
- reconstruction mapping into \(\theta_t\).

The exact implementation remains subordinate to the live method assessment and stage scripts.
