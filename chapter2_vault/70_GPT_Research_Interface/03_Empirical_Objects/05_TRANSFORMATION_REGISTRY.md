# Transformation Registry

| Transformation | Use | Warning |
|---|---|---|
| log | scale and elasticity interpretation | positive series only |
| first difference | integration diagnostics and growth | loses level information |
| lag | temporal ordering | document exact timing |
| accumulated interaction | state-conditioned capital accumulation | may become I(2) |
| deflation | convert nominal to real | requires price-object match |
| normalization | fix level or compare paths | does not identify coefficients |
| annual aggregation | align frequency | document sum/average/end-period rule |
