# Variable Dictionary

## U.S. core

- `y`: log real output.
- `k_Kcap`: log capacity-capital stock.
- `omega` or `tau`: distribution/technique state.
- `q_omega_h1_Kcap`: lag-conditioned accumulated interaction.
- `theta`: reconstructed capacity-building elasticity.
- `yp`: reconstructed log productive capacity.
- `mu`: reconstructed utilization.

## Chile core

The same conceptual objects apply, but source names, boundaries, and external-state variables must be rebuilt from the Chile source panel.
