# Perverse Supply Response

A perverse supply response exists when local output does **not** move positively with local profit incentives in the conventional way.

## In this corpus
The key mechanism is not irrationality. It is **multinational allocation under host-state conflict**.

Higher local profit rates may facilitate:
- dividend extraction,
- reallocation away from the host country,
- vertical integration elsewhere,
- diversification away from the local extractive base.

Lower local profit rates may, under some conditions, induce higher local output if firms need to sustain repatriated profit volumes.
