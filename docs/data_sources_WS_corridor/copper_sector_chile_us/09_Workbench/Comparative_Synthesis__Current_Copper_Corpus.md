# Comparative Synthesis — Current Copper Corpus

## Provisional synthesis
The current corpus already supports a strong relational architecture.

Young shows the US copper industry had to be historically built. Pettengill then shows that this capacity became more than domestic output: it became market, refining, and export centrality. Mamalakis provides the Chile-side export-sector and state relation. Narayan/Reddy adds the US strategic-state layer. Hojman then gives a compact mechanism that explains why local output behavior under multinational control may diverge from standard profit-incentive expectations. Rivas can deepen the measurement layer, but only if forced through the Shaikh/Basu guardrail.

## Core proposition now available
Copper should be treated as a **Chile/US/world-market relation**, not as a Chilean commodity plus external influences.

## Missing lanes still to build
- direct company-specific nodes on Anaconda, Kennecott, Braden, Chuquicamata;
- explicit nationalization / chilenization cluster;
- fuller nineteenth-century Chilean copper export cluster;
- direct links between refining location and appropriation struggles.
