# Zavaleta Comparative-Relational Repo Classifier Pack

This pack is a **distilled operational layer** for repo use. It is not a replacement for the full Zavaleta bundle and it is not a raw source corpus.

Its purpose is to help Codex and human review classify materials for **AR-variable building** without turning the full comparative-relational vault into an undisciplined prompt swamp.

## What this pack is for
- Classifying sources before extraction
- Routing candidate objects through a comparative-relational grammar
- Reducing conceptual drift in AR-variable construction
- Preserving the rule: **source-first, overlay-later**

## What this pack is not for
- Automatic variable generation from theory notes alone
- Treating every Zavaleta note as equally authoritative
- Replacing source verification
- Drafting chapter prose directly

## Recommended repo placement
```text
docs/method_classifier_reference/
└── zavaleta_comparative_relational/
    ├── README.md
    ├── classifier_digest.md
    ├── codex_use_rules.md
    ├── selected_core_manifest.md
    ├── ar_variable_crosswalk_template.csv
    └── codex_prompt_ar_variable_classifier.txt
```

## Minimal workflow
1. Classify the source or candidate object
2. Assign source family and scale
3. Route through operator logic
4. Infer mechanism family
5. Only then propose AR candidate variables
6. Flag translation risk and AR relevance

## Core discipline
Do not let Codex infer variables directly from the whole Zavaleta vault.
Use this pack as a **classifier reference**.
