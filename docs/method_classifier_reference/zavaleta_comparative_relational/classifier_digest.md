# Classifier Digest

## Purpose
This digest translates the comparative-relational toolkit into a usable repo grammar for AR-variable building.

## Core methodological claims
1. The case is a **configuration**, not a container.
2. Comparison is **relational**, not additive.
3. External determination is **constitutive**, not exogenous scenery.
4. Concepts travel only through **historical translation**.
5. Intake should be **source-first, overlay-later**.

## What Codex should actually do with this
Codex should not “apply Zavaleta” as a theme. It should use this toolkit to answer:
- What kind of object is this?
- At what scale does it operate?
- What relation does it mediate?
- Is it structural background, conjunctural trigger, or crisis marker?
- Which operator is actually relevant?
- Is translation risk low, medium, or high?
- Does this belong in AR-variable construction at all?

## Rule for AR-variable construction
Candidate variables are admissible only after:
source classification -> scale identification -> operator routing -> mechanism inference -> evidentiary check.

## Typical failure modes
- treating background Cold War material as immediate variable content
- flattening Chile and the US into symmetric comparison
- using theory notes as if they were evidence
- exporting a concept without historical translation
- proposing variables before identifying the relation they are supposed to track
