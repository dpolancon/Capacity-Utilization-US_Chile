It is an agent-facing operational distillation of the toolkit, not a replacement for the full vault.

## Non-negotiable method rules

1. **Cases are configurations, not containers.**
   Do not begin from "Chile" or "the US" as sealed units. Begin from the historically specific configuration of state form, class capacity, mediation structures, unresolved tasks, and contradiction.

2. **Comparison must be relational, not additive.**
   A comparison is not finished when two cases are placed side by side. State what each case renders visible in the other and what wider field makes both cases intelligible.

3. **External determination is constitutive.**
   Imperial, regional, and transnational power must enter the case internally. Do not add scale as a final context paragraph.

4. **Concepts travel only through reconstruction.**
   If importing a concept or operator from the toolkit into a new archive or writing task, translate it to the new evidence base before applying it.

5. **Strategy must be inferred from structure.**
   Do not moralize actors or rank strategies abstractly. State what was historically thinkable, organizationally supportable, structurally viable, blocked, or misrecognized.

6. **The state is internally differentiated.**
   Distinguish state power from state apparatus before making state-centered claims.

7. **Historical time is layered.**
   Track overlapping temporalities, unresolved tasks, and conjunctural sequences explicitly. Do not use layered time as rhetorical atmosphere.

8. **Retrospective knowledge is not actor knowledge.**
   Preserve the difference between what later analysis can reconstruct and what actors could see from inside the conjuncture.

## Repo-specific application rules

### Chapter 2 and macro-comparative work

- Use the protocol when interpreting the US/Chile comparison, center/periphery asymmetries, Fordism/ISI contrasts, crisis conjunctures, and state-form differences.
- Relate econometric or accounting results back to historically specific configurations rather than treating the national case as a neutral data container.
- When drawing comparative claims from macro outputs, state the shared problematic, the structural difference, and the wider field of determination.

### WS corridor and archive work

- Preserve the repo grammar already in force:
  - dimension = structural field
  - mechanism = causal operator
  - variable = observable trace
  - source = evidentiary anchor
- Use the Zavaleta operators to deepen relational interpretation, not to flatten the corridor into chronology or free-form theory notes.

## Session contract

Before drafting or synthesizing comparative-historical output, declare:

- primary case
- comparative reference case
- historical window
- immediate output target
- operator carrying the main methodological load

## Minimal working sequence

1. Define the case as a configuration, not a country profile.
2. Specify the relational axis linking the focal case to the contrastive case.
3. Build external determination into the configuration from the start.
4. Identify the temporal layering and the active contradiction.
5. Separate structural reconstruction from actor knowledge in the conjuncture.
6. Infer the strategic corridor from the configuration rather than from moral preference.
7. Assemble prose so each analytical paragraph can route backward to both a method operator and an evidence base.

## Drift controls

Stop and correct if the analysis slips into any of these forms:

- comparison without relation
- state talk without differentiation
- temporality as atmosphere
- scale as afterthought
- strategy as moral verdict
- portability without translation

## Output rule

A comparative-relational note, memo, or paragraph block is not finished unless it makes clear:

- what configuration is being analyzed
- what contrastive relation makes the comparison informative
- how external determination enters the case
- which strategic possibilities were opened, narrowed, or blocked
- what evidence anchors the claim
